# Tools

Utility scripts for exporters, converters, and mesh cleanup.

## Structure

```
tools/
├── exporters/           # Scripts to export from Blender/Meshy
├── converters/          # Convert between formats (FBX → RBMX, etc.)
└── mesh-cleanup-scripts/ # Clean up Meshy models before import
```

## Usage

These tools help automate the pipeline:
- Export models from Meshy/Blender
- Convert between file formats
- Clean up mesh issues before importing to Studio
