# Beds

Bed models for the rest system:

- **BlanketBed.model** - Cozy blanket bed
- **BoxBed.model** - Cardboard box bed
- **MoonRockBed.model** - Moon rock bed
- **NebulaHammock.model** - Hammock in nebula
- **StarCradle.model** - Star-shaped cradle

Each bed should have:
- Model with collision
- BedController script
- Rest interaction point
