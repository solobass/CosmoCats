# StarParticles

Star particle effects for the sky.

## Setup Instructions

1. Create a Part in Roblox Studio
2. Name it "StarParticles"
3. Add a ParticleEmitter
4. Configure particles to look like stars (small, twinkling)
5. Set emission rate and lifetime
6. Place in Lighting or as a child of Lighting

- This is a placeholder file
