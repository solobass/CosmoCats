# Lighting Structure

Lighting effects and controllers for the space environment.

## Structure

### DayNightController.lua
Day/night cycle controller for space environment.
- Purpose: Manages lighting changes over time
- Status: Scaffolded

### NebulaSkybox
Nebula-themed skybox for space atmosphere.
- Purpose: Creates space environment visual
- Setup: Create Sky object in Roblox Studio

### Effects/
Visual effects for the space environment:
- **SpaceFog** - Atmospheric fog effect
- **StarParticles** - Star particle effects
- **AuroraBeams** - Aurora beam effects

## Setup Instructions

1. Create DayNightController script in Lighting
2. Create Sky object for NebulaSkybox
3. Create effect objects in Effects/ folder
4. Configure lighting properties in Roblox Studio

## Notes

- DayNightController is a script (scaffolded)
- NebulaSkybox and Effects are Roblox objects (create in Studio)
- All effects should be placed in Lighting service or as children
