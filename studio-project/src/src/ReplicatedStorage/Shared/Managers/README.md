# Managers Folder

Shared manager modules that coordinate between systems.

## Files

- **CatAvatarManager.lua** - Avatar management (exists in parent Shared folder)
- **SpeciesManager.lua** - Species selection/management (handled by Modules/Species)
- **ClanManager.lua** - Clan management (handled by Modules/Clans)

Note: Some managers are in Modules/ for better organization.
