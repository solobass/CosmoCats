# Assets Folder

This folder contains game assets that are accessible to both client and server.

## Structure

- **Models/** - 3D models (species, items, beds, etc.)
  - **Species/** - Species-specific models
  - **Items/** - Item models
  - **Beds/** - Bed models
- **Animations/** - Animation files
  - **Emotes/** - Emote animations
  - **Idle/** - Idle animations
  - **Movement/** - Movement animations
- **Sounds/** - Audio files
  - **Music/** - Background music
  - **SFX/** - Sound effects
- **Textures/** - Custom textures
- **UI/** - UI assets (icons, images)
