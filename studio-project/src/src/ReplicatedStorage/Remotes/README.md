# Remotes Folder

This folder contains RemoteEvents and RemoteFunctions for client-server communication.

## Structure

RemoteEvents and RemoteFunctions are organized by system:
- **PlayerDataRemotes/** - Player data synchronization
- **ShopRemotes/** - Shop purchases
- **InventoryRemotes/** - Inventory management
- **SpeciesRemotes/** - Species selection
- **ClanRemotes/** - Clan operations
- **CharacterRemotes/** - Character customization
- **EmoteRemotes/** - Emote playback
- **BedRemotes/** - Bed placement
- **CurrencyRemotes/** - Currency updates

Note: These are created dynamically by the systems, but this folder structure documents where they should be placed.
