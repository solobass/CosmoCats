# AuroraBeams

Aurora beam effects for the sky.

## Setup Instructions

1. Create Parts or use Beam objects in Roblox Studio
2. Name it "AuroraBeams"
3. Add Beam objects or particle effects
4. Configure for aurora appearance (curved, colorful beams)
5. Set colors (green, blue, purple)
6. Place in Lighting or as a child of Lighting
