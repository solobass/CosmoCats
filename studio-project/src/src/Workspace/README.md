# Workspace Structure

Game world elements, maps, spawn points, and interactable items.

## Structure

### Map/
Game world areas:
- **NebulaForest/** - Forest area in nebula
- **LunarCaverns/** - Cavern system on a moon
- **StarfallDesert/** - Desert with falling stars
- **CometShores/** - Beach area with comets
- **VoidTemple/** - Temple in the void

### SpawnPoints/
Player spawn locations:
- **ForestSpawn** - Spawn in NebulaForest
- **CavernSpawn** - Spawn in LunarCaverns
- **DesertSpawn** - Spawn in StarfallDesert
- **ShoresSpawn** - Spawn in CometShores
- **TempleSpawn** - Spawn in VoidTemple

### Beds/
Bed models for rest system:
- **BlanketBed.model** - Cozy blanket bed
- **BoxBed.model** - Cardboard box bed
- **MoonRockBed.model** - Moon rock bed
- **NebulaHammock.model** - Hammock in nebula
- **StarCradle.model** - Star-shaped cradle

### InteractableItems/
Collectible and interactive items:
- **StardustCrystal.model** - Stardust currency collectible
- **CosmicBerry.model** - Berry collectible
- **CrystalFragment.model** - Crystal fragment collectible

## Setup Instructions

1. Create Models, Parts, or Folders in Roblox Studio
2. Name them exactly as listed above
3. Place them in the appropriate folders
4. Add scripts and properties as needed

## Notes

- All items are placeholders - create actual models in Roblox Studio
- Each folder/item has a README.md with setup instructions
- Models should be created with proper collision and visual elements
