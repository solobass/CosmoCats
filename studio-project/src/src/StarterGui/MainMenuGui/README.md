# MainMenuGui

Main menu UI for Cosmo Cats.

## UI Structure (Create in Roblox Studio)

**MainMenuScreenGui** (ScreenGui)
- **MainFrame** (Frame) - Main container
  - **Title** (TextLabel) - Game title
  - **PlayButton** (TextButton) - Start game
  - **SettingsButton** (TextButton) - Open settings
  - **ShopButton** (TextButton) - Open shop
  - **InventoryButton** (TextButton) - Open inventory

## Scripts

- **Scripts/MainMenu.client.lua** - Main menu controller

## Notes

- Create ScreenGui in Roblox Studio
- Add UI elements as documented above
- Connect buttons to MainMenu.client.lua
