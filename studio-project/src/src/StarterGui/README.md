# StarterGui Structure

UI folders and scripts for Cosmo Cats.

## Structure

Each GUI folder contains:
- A ScreenGui object (created in Roblox Studio)
- A Scripts/ folder with the controller script
- A README.md documenting the UI structure

## GUI Folders

1. **MainMenuGui/** - Main menu interface
2. **SpeciesSelectorGui/** - Species selection interface
3. **CharacterCreatorGui/** - Character customization interface
4. **EmoteBarGui/** - Emote selection bar
5. **ClanGui/** - Clan management interface
6. **CurrencyGui/** - Stardust currency display

## Setup Instructions

1. In Roblox Studio, create ScreenGui objects in each folder
2. Add UI elements (Frames, Buttons, Labels) as documented in each README
3. Connect UI elements to the controller scripts
4. Test each UI system independently

## Notes

- All UI elements should be created in Roblox Studio
- Scripts are ready to connect to UI elements
- Each GUI has a README documenting its structure
