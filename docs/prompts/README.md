# Cursor Prompts

Curated prompts for Cursor to generate code following the Cosmo Cats project structure.

## Usage

1. Open `/studio-project/src` in Cursor
2. Use the prompts from this folder
3. Cursor will create files inside the matching folder structure

## Available Prompts

- **Species Controller** - Generate a new species controller
- **Module Template** - Generate a new module following project patterns
- **UI System** - Generate a new UI system
- **Test File** - Generate test files for modules

## How to Use

Copy a prompt from this folder and paste it into Cursor. Cursor will:
- Generate the appropriate files
- Follow the project structure
- Use consistent naming conventions
- Include proper header comments
